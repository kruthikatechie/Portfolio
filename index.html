Projects

Human Activity Recognition

This project implements a real-time Human Activity Recognition system using a CNN-LSTM model. It processes video frames from the HMDB-51 dataset or a webcam to classify activities like walking, running, and jumping. The system demonstrates end-to-end data preprocessing, model training, and live action recognition with visual feedback.

Food Recognition

This project implements a real-time food recognition and calorie estimation system using a YOLOv8 model. It detects food items from uploaded images and retrieves their nutritional information from OpenFoodFacts or the USDA database. The system demonstrates end-to-end image processing, object detection, and nutrition data integration..
